# Lain42 — 配置分享平台 设计文档

**日期**: 2026-06-04  
**状态**: MVP 设计  
**概述**: 类似 Pinterest/Seelen.io 的图片优先配置分享平台。用户分享 Arch Linux ricing、Windows 美化、C/C++ 开发环境、书源等各类配置文件。支持 Web 浏览 + Rust TUI 客户端自动同步。

---

## 1. 拓扑

```
                    ┌──────────────┐
                    │   Nginx:80   │
                    └──┬───┬───┬──┘
           /api/*     /     │     /images/*
              │       │     │        │
     ┌────────▼──┐  ┌─▼─────▼──┐  ┌─▼──────────┐
     │ FastAPI    │  │ SPA 静态 │  │ 图片文件   │
     │ :8000 REST │  │ 文件     │  │ 系统       │
     │ :50051 gRPC│ └──────────┘  └────────────┘
     └──┬────┬────┘        ▲
        │    │    gRPC     │
   ┌────▼────▼──┐  ┌───────┴──────┐
   │ PostgreSQL │  │ Rust TUI CLI │
   │ :5432      │  │ (ratatui)    │
   └────────────┘  └──────────────┘
```

- FastAPI 和 gRPC 同进程运行（HTTP :8000 + gRPC :50051）
- gRPC 客户端直连 50051 端口（不加 Nginx 反向代理）
- Nginx 反向代理 REST API、静态文件 serve 和图片 serve
- 部署: Docker Compose 单机，目标服务器 `myserver`

## 2. 技术栈

| 层 | 选型 | 理由 |
|---|------|------|
| 后端 REST | FastAPI (Python 3.14+) | 异步、类型注解、生态好 |
| 后端 gRPC | grpcio + grpc.aio | 同进程双端口、低摩擦 |
| ORM | SQLAlchemy 2.0 (async) | 异步原生支持 |
| 数据库 | PostgreSQL 17 | JSONB、全文搜索、GIN 索引 |
| Proto 共享 | Monorepo `proto/` 目录 | 两端共用、CI 检查 |
| 前端框架 | Vite + React 18 SPA | 纯 CSR、前后端分离、轻量 |
| UI 组件 | shadcn/ui (Tailwind) | 高质量、可定制 |
| 前端路由 | react-router-dom v7 | SPA 标准 |
| 数据获取 | TanStack Query v5 | 缓存、重取、乐观更新 |
| 全局状态 | Zustand | 轻量、当前用户等全局态 |
| 瀑布流 | react-virtuoso | 虚拟滚动 + masonry + 无限加载 |
| 代码编辑 | Monaco Editor | 配置文件语法高亮编辑 |
| Markdown | react-markdown + rehype-highlight | 教程正文渲染 |
| 图片处理 | Pillow → WebP | 3 级缩略图 (300/600/original) |
| 搜索 | PostgreSQL 全文搜索 (pg_jieba) | 零额外服务、中文分词 |
| 客户端 | Rust + ratatui + tonic | TUI、单二进制、gRPC 原生 |
| 部署 | Docker Compose | Nginx + FastAPI + PostgreSQL + 图片卷 |

## 3. 认证

- **OAuth 提供商**: GitHub / Gitee / GitCode（Provider 抽象层，方便扩展）
- **Token**: JWT (access + refresh)
- **流程**: 前端 OAuth 弹窗 → 后端回调 `/api/auth/login` → 签发 JWT → 前端存 localStorage
- **客户端**: JWT 存在本地 `~/.config/lain/auth.json`，gRPC metadata 携带
- 不做邮箱/密码注册，全部 OAuth 登录

## 6. 图片规则

- 每帖 1-5 张截图
- 第 1 张自动设为封面
- 前端上传 → 后端 Pillow 压缩为 WebP → 生成 3 级尺寸
- 单张 ≤ 2MB
- 路径: `/data/images/{post_id}/{size}_{uuid}.webp`
- Nginx 直接 serve `/data/images/` 目录

## 7. REST API

### 认证
```
GET    /api/auth/providers          # 列出 OAuth 提供商
POST   /api/auth/login              # OAuth 回调 (provider + code → JWT)
POST   /api/auth/refresh            # 刷新 access token
GET    /api/auth/me                 # 当前用户
```

### 帖子
```
GET    /api/posts                   # 分页 (?category=&tag=&sort=latest|popular&page=&per_page=)
GET    /api/posts/:id               # 详情
POST   /api/posts                   # 发布 (auth required)
PUT    /api/posts/:id               # 编辑 (owner only)
DELETE /api/posts/:id               # 删除 (owner only)
```

### 用户
```
GET    /api/users/:username         # 用户主页
GET    /api/users/:username/posts   # 用户的帖子
GET    /api/users/:username/collections  # 公开收藏夹
```

### 收藏夹 (auth required)
```
GET    /api/collections                   # 我的收藏夹
POST   /api/collections                   # 创建
PUT    /api/collections/:id               # 编辑
POST   /api/collections/:id/posts/:pid    # 收藏帖子
DELETE /api/collections/:id/posts/:pid    # 取消收藏
```

### 点赞 (auth required)
```
POST   /api/posts/:id/like                # 点赞/取消 (toggle)
```

### 搜索
```
GET    /api/search?q=&category=&sort=latest&page=&per_page=
```

### 上传 (auth required)
```
POST   /api/upload/image                  # 上传图片 → 返回 URLs
```

## 8. gRPC 服务

```protobuf
syntax = "proto3";

service ConfigSync {
  rpc ListSubscriptions(Empty) returns (SubscriptionsResponse);
  rpc GetCollectionConfigs(CollectionRequest) returns (ConfigsResponse);
  rpc PullConfig(ConfigRequest) returns (ConfigArchive);
  rpc CheckUpdates(VersionCheckRequest) returns (UpdateStatus);
}
```

- 客户端本地维护 `~/.config/lain/state.json`，记录 `config_id → version_hash`
- `CheckUpdates` 对比远端/本地版本 → 返回有更新的配置列表
- `PullConfig` 拉取单个完整配置的所有文件和元数据

## 9. Rust TUI 客户端

### 技术
- ratatui (TUI) + tokio (async) + tonic (gRPC)

### 功能 (MVP: 同步优先)
- 启动 → 读取本地 auth token → 连接 gRPC → CheckUpdates
- 显示远端/本地版本对比
- 一键 Apply：拉取配置文件 → 写入本地路径
- Diff 预览模式（可选，看配置变更了什么）

### TUI Tab
- **Sync**: 订阅的配置列表、版本对比、一键同步
- **My Configs**: 查看/管理自己发布的配置
- **Settings**: 服务器地址、OAuth 登录、同步策略

### 分发
- 单二进制编译 (Linux/macOS/Windows)
- Nginx 托管下载 (`/download/lain-linux-amd64`)

## 12. 部署 (Docker Compose)

```yaml
services:
  db:
    image: postgres:17
    volumes:
      - pgdata:/var/lib/postgresql/data
    environment:
      POSTGRES_DB: lain42
      POSTGRES_USER: lain42
      POSTGRES_PASSWORD: ${DB_PASSWORD}

  app:
    build: ./backend
    ports:
      - "8000:8000"
      - "50051:50051"
    volumes:
      - images:/data/images
    depends_on:
      - db
    environment:
      DATABASE_URL: postgresql+asyncpg://...
      JWT_SECRET: ${JWT_SECRET}
      GITHUB_CLIENT_ID: ${GITHUB_CLIENT_ID}
      GITHUB_CLIENT_SECRET: ${GITHUB_CLIENT_SECRET}
      GITEE_CLIENT_ID: ${GITEE_CLIENT_ID}
      GITEE_CLIENT_SECRET: ${GITEE_CLIENT_SECRET}
      GITCODE_CLIENT_ID: ${GITCODE_CLIENT_ID}
      GITCODE_CLIENT_SECRET: ${GITCODE_CLIENT_SECRET}

  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf
      - ./frontend/dist:/usr/share/nginx/html
      - images:/usr/share/nginx/html/images

volumes:
  pgdata:
  images:
```
